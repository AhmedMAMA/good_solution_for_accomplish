# Objectif du projet
Réalisation de solution embarqué se servant de model IA pour assurer des tâches spécifique par vision par ordinateur.

# Structure du projet



Ensemble de code création dans plusieurs language de programmation accéptable en embarqué répartie comme suite
  Dossier | Language | Rôle
  --|--|--
  ``python`` | Python | Testing et potentiellement embarquement avec mise en place de service pour de l'automatisation
  ``bases/math`` | C/C++ | Pour du full embarquable avec traitement par niveau pour du temps réle
  ``model`` | Onnx | Modèles à exploiter
  ``report`` | **markdown** | Prise de notes (Documentation et compte rendu)
